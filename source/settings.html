Kodi Unity Style Menu Settings

Use a 'whitelist' to a define a list of websites you want the extension to run on. If no whitelist is used the extension will run on all sites.

List the website(s) that the menu will appear on separated by a comma. Enter the hostname only, eg www.netflix.com, not https://www.netflix.com/au

Set the menu colour as a hexadecimal value: AABBCC (R|G|B) .

This extension is designed to compliment the excellent Unity skin for Kodi, when launching chrome in fullscreen on a TV or dedicated media PC.
